Simulate Data and Run Linear Regression Here’s how to simulate some basic data and run a regression: set.seed(123) n <- 100 x1 <- rnorm(n,0,1) x2 <- rnorm(n,3,5) error <- rnorm(n,0,10) y <- x1 + x2 + error model <- lm(y~x1+x2) summary(model) vcov(model) One thing to pay attention to are the last two lines. »
